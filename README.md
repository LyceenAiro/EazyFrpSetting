# EazyFrpSetting
为一些不明白Frp运行原理的新人编写的一款快速入门的Frp客户端  
-GitHub:https://github.com/LyceenAiro/EazyFrpSetting
  
需要使用该软件配置Frp首先你需要Frp的软件本体  
-GitHub:https://github.com/fatedier/frp  
-需求文件:frpc.exe  
-本软件暂不支持server端快速搭建
